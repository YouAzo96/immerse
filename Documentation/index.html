Introduction

Thank you for purchasing Chatvia - React js Responsive Chat App Template.

Chatvia is built with Bootstrap 5.3.0 and React Js 18, SCSS with responsive with all devices and supported with Dark, light, RTL modes. You can change mode very quickly by doing a single change. It has many features like one to one chat, group chat, contact, send files, online users, read and unread new messages from users, authentication pages and many more.
